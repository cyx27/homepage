4 HTML5基础

November 20, 2022

1. 什么是HTML5

  1. HTML 的最新稳定版本， HTML5 将 HTML 从⽤于构造⼀个⽂档的⼀个简单标记，扩展到⼀个完整的应⽤程序开发平台。除其他功能外，HTML5 还包括新元素和⽤于增强存储、多媒体和硬件访问的JavaScript APIs 。
  2. HTML5 是 W3C 与 WHATWG 合作的结果,WHATWG 指 Web Hypertext ApplicationTechnology Working Group。

2. HTML5应用

  1. 本地数据存储
  2. 访问本地文件
  3. 本地SQL数据
  4. 缓存引用
  5. Javascript工作者
  6. XHTMLHttpRequest2

3. HTML5优势

  1. 跨浏览器兼容性
  2. 大道至简
    1. 旧路新铺
    2. 简化的Doctype
    3. 简化的字符集
    4. 简化的标签
  3. 语义化
    1. 根据结构化的内容选择合适的标签
  4. 可用性和用户体验的改进
    1. 无障碍
  5. 无插件范式
    1. 过去，很多功能只能通过插件或者复杂的hack(本地绘图API、本地socket等)来实现，但是在HTML5中提供了对这些功能的原⽣⽀持。
  6. 安全
    1. 跨源资源共享(CORS)使Web服务器允许其他域名的网页访问自己的资源
    2. CORS放宽了同源访问规则(Same Origin Rule)，这是Web浏览器内置的基础安全措施之⼀。
  7. 大量用于移动应用和游戏
    1. 多设备跨平台
    2. 自适应网页设计
  8. 在网站抓取和索引方面， HTML5 对 SEO 友好

4. HTML5新功能特性

  1. 新元素

  1. 新属性
  2. 完全支持CSS3
  3. Video和Audio
  4. 2D/3D制图
    1. SVG是一种用XML描述2D图形的语言，可以为元素附加JavaScript事件处理程序
    2. Canvas绘制2D图像，使用Javascript，是⼀个像素⼀个像素渲染的

  1. web存储

    1. 早前本地存储使⽤的是 cookie，但是Web 存储更加的安全与快速
    2. 客户端存储数据的两个对象为：
      1. localStorage - 用于长久保存整个网站的数据，保存的数据没有过期时间，直到手动去除。
      2. sessionStorage - 用于临时保存同一窗口（或标签页）的数据，在关闭窗口或标签页之后将会删除这些数据。

  2. 离线浏览

  3. HTML5 Web SQL数据库

    三个核心方法：

    1. openDatabase：这个方法使用现有的数据库或者 新建的数据库创建⼀个数据库对象。
    2. transaction：这个方法让我们能够控制⼀个事务， 以及基于这种情况执行提交或者回滚。
    3. executeSql：这个方法用于执行实际的 SQL 查询。
